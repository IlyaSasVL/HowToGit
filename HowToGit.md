# Подсказка по Git
### Создание репозитория
```sh
git init
```
### Добавление содержимого рабочего каталога в индекс
```sh
git add
```
### Создание коммита
```sh
git commit -m "Massage"
```
### Просмотр истории коммитов
```sh
git log
```
### Вывод истории коммитов в кратком виде
```sh
git log --oneline
```
### Перемещение между коммитами
```sh
git checkout
```
